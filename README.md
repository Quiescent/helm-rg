# helm-rg

Helm frontend for `rg` ([ripgrep](https://github.com/BurntSushi/ripgrep)).

## Quickstart

```emacs-lisp
(add-to-list 'load-path "path/to/helm-rg/")
(require 'helm-rg)
```

## Commands

- `helm-rg`
- `helm-rg-at-point`
- `helm-projectile-rg`
- `helm-projectile-rg-at-point`

## Customisation

`M-x customize-group <RET> helm-rg`
